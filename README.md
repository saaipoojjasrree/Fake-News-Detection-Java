# Fake News Detection System (Java)

A simple Java-based mini project that detects fake news using keyword-based text analysis.

## Features
- Takes user input news text
- Uses predefined fake keywords
- Classifies news as Real or Fake

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac FakeNewsDetection.java
2. Run:
   java FakeNewsDetection
